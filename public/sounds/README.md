# Notification Sounds

Thêm file `notification.mp3` vào folder này để phát âm thanh thông báo khi có tin nhắn mới.

## Yêu cầu

- Định dạng: MP3
- Tên file: `notification.mp3`
- Độ dài khuyến nghị: 1-3 giây
- Âm lượng vừa phải

## Nguồn tải

Bạn có thể tải file âm thanh miễn phí từ:

- https://notificationsounds.com/
- https://freesound.org/
- https://mixkit.co/free-sound-effects/notification/

## Lưu ý

Nếu không có file âm thanh, hệ thống sẽ bỏ qua phần phát âm thanh mà không gây lỗi.
